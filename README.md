## ✨ Features
- 👤 User system  
- 🍝 Create & share recipes  
- ❤️ Like recipes  
- 💬 Comment system  
- ⭐ Rating system  
- 🔍 Search by ingredients  
- 📰 Personalized feed  

---

## 🛠 Tech Stack
- Frontend: React (Vite)  
- Backend: Node.js, Express  
- Database: MongoDB Atlas  
- Deployment: Vercel + Render  

---

## ⚙️ Setup

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm run dev
